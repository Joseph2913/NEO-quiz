Setup Guide

Follow these steps to get the NEO Quiz Automation tool running on your machine. Copy the prompt below and paste it into your terminal or Claude Code session.

1

Prerequisites

Make sure you have Node.js (v18 or later) installed. Download it from https://nodejs.org if needed. You can verify by opening a terminal and running:

node --version
2

Clone the Repository

Open a terminal and run:

git clone https://github.com/YOUR_USERNAME/neo-quiz-automation.git cd neo-quiz-automation
3

Install Dependencies & Start

Run these two commands:

npm install npm start

The app will open automatically in your browser at http://localhost:3000.

4

First-Time Sign In

The first time you process a form, a Chromium browser will open and ask you to sign in to your Microsoft account. After signing in once, your session is saved locally in auth-state.json so you won't need to sign in again.

Full Setup Prompt (copy & paste into terminal)

This single block of commands will clone, install, and start the tool:
