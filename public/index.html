Setup Guide

Claude Code Setup Prompt

Copy this prompt, paste it into Claude Code, and it will do everything for you.

Set up the NEO Quiz Automation tool on my machine. This tool automates Microsoft Forms quiz creation for the LUSA NEO training program. Please complete every step below automatically — I should not need to do anything manually.

## Step 1: Check and install Node.js if needed

Check if Node.js (v18+) is installed by running `node --version`. If it is NOT installed or the version is below 18:
- On macOS: Install Homebrew first if not present (`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`), then run `brew install node`
- On Windows: Download and run the Node.js installer from https://nodejs.org using `winget install OpenJS.NodeJS.LTS` or guide me to download it manually if winget is unavailable
- On Linux: Run `curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash - && sudo apt-get install -y nodejs`

Verify the installation succeeds before continuing.

## Step 2: Check and install Git if needed

Check if Git is installed by running `git --version`. If it is NOT installed:
- On macOS: Run `xcode-select --install` or `brew install git`
- On Windows: Run `winget install Git.Git`
- On Linux: Run `sudo apt-get install -y git`

## Step 3: Clone the repository

Run these commands:
```
git clone https://github.com/Joseph2913/NEO-quiz.git
cd NEO-quiz
```

## Step 4: Install dependencies

Run `npm install` to install Express and Playwright.

Then run `npx playwright install chromium` to download the Chromium browser that the automation uses.

## Step 5: Start the application

Run `npm start` to start the server. It will automatically open the app in the browser at http://localhost:3000.

If the browser does not open automatically, open http://localhost:3000 manually.

## After setup is complete

Tell me: "Setup complete! The NEO Quiz Automation tool is running at http://localhost:3000. You can now use the app in your browser."

Also tell me these usage tips:
- To process a SINGLE form: Click a quiz in the left sidebar, review the questions, click "Approve & Continue", paste the Microsoft Forms editor URL, and click "Start Automation"
- To BULK process: On the Dashboard, paste form names and URLs (comma or tab separated, one per line) into the Bulk Process box, click "Parse & Review", then "Start Bulk Processing"
- The first time you run an automation, a Chromium browser will open asking you to sign in to your Microsoft account. After signing in once, the session is saved so you won't need to sign in again.
- To stop the server later, press Ctrl+C in the terminal where it is running. To restart, run `npm start` again from the NEO-quiz folder.
How to use the prompt above

You don't need to take any technical action yourself. Just follow these 4 steps and the prompt above handles everything else:

1

Download VS Code

If you don't already have it, download Visual Studio Code from https://code.visualstudio.com and install it.

2

Install the Claude Code Extension

Open VS Code, go to the Extensions panel (the square icon on the left sidebar), search for "Claude Code", and click Install.

3

Connect Your Claude Account

Open the Claude Code panel in VS Code and sign in with your Claude / Anthropic account. If you don't have one, you can create one at https://claude.ai.

4

Paste the Prompt

Click the "Copy to Clipboard" button above, then paste the prompt into the Claude Code chat. Claude will automatically install everything, clone the project, and start the app for you. That's it.
